## 1. Voice Processing Architecture

### 1.1 What is Voice-to-Action?

**Voice-to-Action** is the bridge between human intent (spoken commands) and robot execution. It transforms natural language speech into structured, actionable instructions that downstream systems can interpret and execute.

In the context of a humanoid robot, the voice processing pipeline consists of:

```
┌─────────────────┐
│   Audio Input   │
│  (Microphone)   │
└────────┬────────┘
         │
         ↓
┌─────────────────────────────┐
│  Audio Pre-Processing       │
│  • Normalize volume          │
│  • Detect silence periods    │
│  • Validate duration (0.5-   │
│    300 seconds)             │
└────────┬────────────────────┘
         │
         ↓
┌─────────────────────────────┐
│  OpenAI Whisper API         │
│  • Send audio to cloud      │
│  • Receive transcript text  │
│  • Get confidence score     │
└────────┬────────────────────┘
         │
         ↓
┌─────────────────────────────┐
│  Post-Processing            │
│  • Confidence thresholding   │
│  • Error classification      │
│  • Logging & retry logic     │
└────────┬────────────────────┘
         │
         ↓
┌─────────────────────────────┐
│  ROS 2 Publication           │
│  • Publish to /voice/...    │
│  • Topic: std_msgs/String   │
│  • Latency: &lt;2 seconds      │
└─────────────────────────────┘
```

**Key Concepts**:

1. **Whisper Model**: A robust speech-to-text model trained on 680,000 hours of multilingual audio. It supports 99 languages and is remarkably resilient to accents, background noise, and technical language.

2. **Confidence Scoring**: Whisper doesn't explicitly return confidence, but we infer it from:
   - Decoding score (probability of the transcribed sequence)
   - Comparison with alternative transcriptions
   - Consistency across repeated submissions

3. **Latency Considerations**:
   - Network round-trip: 0.1-1.0 second
   - Whisper processing: 0.5-2 seconds (depends on audio length)
   - ROS 2 overhead: &lt;100 milliseconds
   - **Target**: Complete transcription within 2 seconds

4. **Error Cases**:
   - **Audio Too Short**: &lt;0.5 seconds (silence or incomplete utterance)
   - **Audio Too Long**: &gt;300 seconds (memory constraints)
   - **API Rate Limit**: Too many requests in short time (retry with backoff)
   - **API Error**: OpenAI service unavailable or invalid key
   - **Low Confidence**: Transcription quality below threshold

## 2. OpenAI Whisper API Setup & Authentication

### 2.1 API Fundamentals

**OpenAI Whisper API** provides a RESTful endpoint for speech-to-text:

```
POST https://api.openai.com/v1/audio/transcriptions
```

**Request Format**:
```python
{
  "model": "whisper-1",               # Fixed model name
  "file": &lt;binary_audio_data&gt;,        # WAV, MP3, M4A, FLAC (max 25MB)
  "language": "en",                   # Optional; auto-detects if omitted
  "prompt": "Technical robot commands", # Optional; helps with context
  "response_format": "json"           # json or text
}
```

**Response Format**:
```json
{
  "text": "bring me the red cube from the table",
  "language": "en"
}
```

## 4. ROS 2 Integration: Lifecycle Nodes & Publishers

### 4.1 ROS 2 Lifecycle Management

The **whisper_transcriber_node** uses ROS 2 **lifecycle nodes** to manage state transitions safely:

```
┌─────────────┐
│ unconfigured│
└──────┬──────┘
       │ (configure)
       ↓
┌─────────────┐
│  inactive   │
└──────┬──────┘
       │ (activate)
       ↓
┌─────────────┐
│   active    │ ← Process voice commands here
└──────┬──────┘
       │ (deactivate)
       ↓
┌─────────────┐
│  inactive   │
└──────┬──────┘
       │ (cleanup)
       ↓
┌─────────────┐
│ unconfigured│
└─────────────┘
```

**Lifecycle Benefits**:
- **Safety**: Ensure node is ready before accepting commands
- **Debugging**: Clear state transitions for troubleshooting
- **Recovery**: Restart by cycling through states
- **Testability**: Predictable startup/shutdown behavior

### 4.2 Node Structure

**transcriber_node.py** implements:

1. **on_configure()**: Initialize Whisper client, setup publisher
2. **on_activate()**: Start listening for audio (if live input), create audio stream
3. **on_process_data()**: Handle transcription requests (called by timer or service)
4. **on_deactivate()**: Stop listening, close audio stream
5. **on_cleanup()**: Release resources, close Whisper client

### 4.3 Publishing to ROS 2 Topics

**Topic Definition**:
```yaml
Topic: /voice/transcribed_command
Type: std_msgs/String
Publish Rate: On-demand (after each transcription)
```

**Example Publisher Code**:
```python
from rclpy.node import Node
from std_msgs.msg import String

class WhisperTranscriberNode(Node):
    def __init__(self):
        super().__init__('whisper_transcriber_node')
        self.publisher = self.create_publisher(
            String,
            '/voice/transcribed_command',
            qos_profile=10  # Queue size
        )

    def publish_command(self, transcript: str):
        msg = String()
        msg.data = transcript
        self.publisher.publish(msg)
        self.get_logger().info(f"Published: {transcript}")
```

## 5. Error Handling & Confidence Scoring

### 5.1 Error Classification

| Error Code | Cause | Recovery Strategy |
|-----------|-------|-------------------|
| **AUDIO_TOO_SHORT** | Duration < 0.5s | Prompt user to speak longer |
| **AUDIO_TOO_LONG** | Duration > 300s | Split audio or reject |
| **API_RATE_LIMIT** | &gt;3 requests/minute | Exponential backoff (retry 3x) |
| **API_ERROR** | Service unavailable | Log error, alert user, retry later |
| **INVALID_API_KEY** | Missing/expired key | Check `.env`, regenerate key |
| **NETWORK_ERROR** | No internet connection | Queue request, retry when online |
| **LOW_CONFIDENCE** | Quality < threshold | Ask user to repeat or clarify |

### 5.2 Confidence Thresholding

Whisper doesn't provide explicit confidence scores, but we can estimate by:

1. **Decoding Score**: Probability of transcribed sequence (higher = more confident)
2. **Character Coverage**: Ratio of recognized characters to total
3. **Known Commands**: Cross-reference against known valid commands (if using fixed vocabulary)

**Threshold Policy**:
```python
confidence_threshold = 0.5  # Configurable

if confidence < threshold:
    action = "reject"  # Reject and ask user to repeat
else:
    action = "accept"  # Publish to ROS 2 topic
```

### 5.3 Retry Logic & Exponential Backoff

**Exponential Backoff Pattern** (for API rate limits):

```python
def retry_with_backoff(func, max_attempts=3):
    for attempt in range(max_attempts):
        try:
            return func()
        except RateLimitError as e:
            wait_time = 2 ** attempt  # 1s, 2s, 4s
            if attempt < max_attempts - 1:
                time.sleep(wait_time)
                continue
            raise e
```

## 6. Performance Optimization & Latency Measurement

### 6.1 Latency Breakdown

For a typical 2-second command ("move forward"), latency breakdown:

| Component | Time | % of Total |
|-----------|------|-----------|
| Audio capture | 2.0s | 43% |
| Network upload | 0.2s | 4% |
| Whisper processing | 1.2s | 26% |
| API latency (network) | 0.3s | 6% |
| ROS 2 publish | 0.05s | 1% |
| Buffer/overhead | 0.25s | 5% |
| **Total** | **4.0s** | **100%** |
| **Goal** | **&lt;2s** | — |

To meet the **&lt;2 second goal**, we focus on:
1. Minimize audio capture time (shorter commands)
2. Batch processing (multiple requests together)
3. Parallel execution (transcribe while recording)

### 6.2 Batch Processing

Instead of transcribing one command at a time, batch multiple voice inputs:

```python
# Sequential (slow, ~2s per command)
for command in commands:
    transcript = whisper_client.transcribe(command)  # 2s each

# Batch (faster, ~0.5s amortized per command)
transcripts = whisper_client.batch_transcribe(commands)  # 1s total
```

### 6.3 Metrics Collection

Create **PerformanceMonitor** class to track:

```python
class PerformanceMonitor:
    def __init__(self):
        self.latencies = []        # Per-request latency
        self.accuracies = []       # Transcription accuracy
        self.error_count = 0       # Failed requests
        self.start_time = None

    def record_latency(self, start: float, end: float):
        latency = end - start
        self.latencies.append(latency)
        return latency

    def get_statistics(self):
        return {
            "p50_latency": percentile(self.latencies, 50),
            "p95_latency": percentile(self.latencies, 95),
            "p99_latency": percentile(self.latencies, 99),
            "avg_accuracy": mean(self.accuracies),
            "error_rate": self.error_count / len(self.latencies)
        }
```

            file=audio_file,
            language="en"
        )

        latency = time.time() - start_time

        "transcript": response.text,
        "language": response.get("language", "en"),

### Example 2: ROS 2 Publisher Integration

```python
"""
🚀 Module 4 Example 2: ROS 2 Whisper Publisher
Demonstrates ROS 2 node that publishes transcribed voice commands.
Captures audio from microphone → Whisper API → /voice/transcribed_command topic
"""

import os
import time
from rclpy.node import Node
from rclpy.lifecycle import Node as LifecycleNode
from std_msgs.msg import String
from openai import OpenAI
from dotenv import load_dotenv

class WhisperPublisherNode(LifecycleNode):
    """ROS 2 Lifecycle Node for voice command publishing."""

    def __init__(self):
        super().__init__('whisper_publisher_node')
        self.publisher = None
        self.openai_client = None
        self.recording = False

    def on_configure(self, state):
        """Configure state: Initialize OpenAI client and publisher."""
        self.get_logger().info("🔧 Configuring node...")

        # ✅ Load API key
        load_dotenv()
        api_key = os.getenv("OPENAI_API_KEY")
        self.openai_client = OpenAI(api_key=api_key)

        # ✅ Create publisher
        self.publisher = self.create_publisher(
            String,
            '/voice/transcribed_command',
            qos_profile=10
        )

        self.get_logger().info("✅ Configuration complete")
        return super().on_configure(state)

    def on_activate(self, state):
        """Activate state: Start listening."""
        self.get_logger().info("▶️  Activating node...")
        self.recording = True
        return super().on_activate(state)

    def publish_transcription(self, audio_file_path: str):
        """
        Transcribe audio and publish to ROS 2 topic.

        Args:
            audio_file_path: Path to audio file
        """
        if not self.recording or self.publisher is None:
            self.get_logger().error("❌ Node not active or publisher not initialized")
            return

        try:
            # ✅ Transcribe
            with open(audio_file_path, "rb") as audio_file:
                start_time = time.time()

                response = self.openai_client.audio.transcriptions.create(
                    model="whisper-1",
                    file=audio_file,
                    language="en"
                )

                latency = time.time() - start_time

            transcript = response.text

            # ✅ Publish to topic
            msg = String()
            msg.data = transcript
            self.publisher.publish(msg)

            self.get_logger().info(
                f"✅ Published: '{transcript}' (latency: {latency:.2f}s)"
            )

        except Exception as e:
            self.get_logger().error(f"❌ Error: {str(e)}")

    def on_deactivate(self, state):
        """Deactivate state: Stop listening."""
        self.get_logger().info("⏸️  Deactivating node...")
        self.recording = False
        return super().on_deactivate(state)

    def on_cleanup(self, state):
        """Cleanup state: Release resources."""
        self.get_logger().info("🧹 Cleaning up...")
        self.publisher = None
        self.openai_client = None
        return super().on_cleanup(state)


def main():
    """Example: Launch node and publish transcription."""
    import rclpy

    rclpy.init()
    node = WhisperPublisherNode()

    # ✅ Transition to active state
    node.trigger_transition(1)  # Configure
    time.sleep(0.5)
    node.trigger_transition(3)  # Activate
    time.sleep(0.5)

    # ✅ Test with audio file
    audio_file = "examples/data/test_audio.wav"
    if os.path.exists(audio_file):
        node.publish_transcription(audio_file)

    # ✅ Cleanup
    node.trigger_transition(4)  # Deactivate
    time.sleep(0.5)
    node.trigger_transition(5)  # Cleanup

    node.destroy_node()
    rclpy.shutdown()


if __name__ == "__main__":
    main()
```

### Example 3: Error Handling & Low Confidence Rejection

```python
"""
🚀 Module 4 Example 3: Error Handling and Confidence Thresholding
Demonstrates graceful error handling for:
- Low confidence predictions
- Audio validation failures
- API errors and timeouts
"""

import os
import time
from openai import OpenAI
from dotenv import load_dotenv

class ConfidenceBasedTranscriber:
    """Transcriber with confidence scoring and error handling."""

    def __init__(self, confidence_threshold: float = 0.5, max_retries: int = 3):
        """
        Initialize transcriber with configuration.

        Args:
            confidence_threshold: Minimum confidence [0.0-1.0]
            max_retries: Max retry attempts for API errors
        """
        load_dotenv()
        self.api_key = os.getenv("OPENAI_API_KEY")
        self.client = OpenAI(api_key=self.api_key)
        self.confidence_threshold = confidence_threshold
        self.max_retries = max_retries
        self.error_count = 0

    def validate_audio(self, audio_bytes: bytes) -> tuple[bool, str]:
        """
        ✅ Validate audio file constraints.

        Returns:
            (is_valid, error_message)
        """
        # Check minimum size (0.5 seconds at 16kHz 16-bit mono = ~16KB)
        min_size = 16000 * 0.5 * 2  # 16kHz * 0.5s * 2 bytes
        if len(audio_bytes) < min_size:
            return False, "AUDIO_TOO_SHORT: duration < 0.5 seconds"

        # Check maximum size (300 seconds = ~9.6MB)
        max_size = 16000 * 300 * 2
        if len(audio_bytes) > max_size:
            return False, "AUDIO_TOO_LONG: duration > 300 seconds"

        return True, ""

    def transcribe_with_retry(self, audio_bytes: bytes, language: str = "en") -> dict:
        """
        ✅ Transcribe audio with exponential backoff retry logic.

        Returns:
            {
                'status': 'success' | 'error' | 'low_confidence',
                'transcript': str or None,
                'confidence': float,
                'error': str or None,
                'attempts': int
            }
        """
        # ✅ Validate audio first
        is_valid, error_msg = self.validate_audio(audio_bytes)
        if not is_valid:
            self.error_count += 1
            return {
                'status': 'error',
                'transcript': None,
                'confidence': 0.0,
                'error': error_msg,
                'attempts': 0
            }

        # ✅ Retry logic with exponential backoff
        for attempt in range(self.max_retries):
            try:
                start_time = time.time()

                response = self.client.audio.transcriptions.create(
                    model="whisper-1",
                    file=("audio.wav", audio_bytes),
                    language=language
                )

                latency = time.time() - start_time
                transcript = response.text

                # ✅ Estimate confidence (simplified: use text length as proxy)
                confidence = min(1.0, len(transcript) / 50)  # Heuristic

                # ❌ Check if confidence below threshold
                if confidence < self.confidence_threshold:
                    self.error_count += 1
                    return {
                        'status': 'low_confidence',
                        'transcript': transcript,
                        'confidence': confidence,
                        'error': f"Low confidence: {confidence:.2f} < {self.confidence_threshold}",
                        'attempts': attempt + 1
                    }

                # ✅ Success
                return {
                    'status': 'success',
                    'transcript': transcript,
                    'confidence': confidence,
                    'error': None,
                    'attempts': attempt + 1
                }

            except Exception as e:
                error_str = str(e)

                # ⚠️ Rate limit error: exponential backoff
                if "rate_limit" in error_str.lower():
                    if attempt < self.max_retries - 1:
                        wait_time = 2 ** attempt
                        print(f"⚠️  Rate limit hit. Retrying in {wait_time}s...")
                        time.sleep(wait_time)
                        continue

                # ❌ Other errors: return failure
                self.error_count += 1
                return {
                    'status': 'error',
                    'transcript': None,
                    'confidence': 0.0,
                    'error': str(e),
                    'attempts': attempt + 1
                }

        # ❌ All retries exhausted
        self.error_count += 1
        return {
            'status': 'error',
            'transcript': None,
            'confidence': 0.0,
            'error': "Max retries exceeded",
            'attempts': self.max_retries
        }

    def get_statistics(self) -> dict:
        """Get error statistics."""
        return {
            'total_errors': self.error_count,
            'confidence_threshold': self.confidence_threshold
        }


if __name__ == "__main__":
    # ✅ Example usage
    transcriber = ConfidenceBasedTranscriber(confidence_threshold=0.5)

    # ✅ Test with a real audio file
    audio_path = "examples/data/test_audio.wav"

    if os.path.exists(audio_path):
        with open(audio_path, "rb") as f:
            audio_bytes = f.read()

        result = transcriber.transcribe_with_retry(audio_bytes)

        print(f"Status: {result['status']}")
        print(f"Transcript: {result['transcript']}")
        print(f"Confidence: {result['confidence']:.2f}")
        if result['error']:
            print(f"Error: {result['error']}")
        print(f"Attempts: {result['attempts']}")
    else:
        print(f"❌ Audio file not found: {audio_path}")
```
