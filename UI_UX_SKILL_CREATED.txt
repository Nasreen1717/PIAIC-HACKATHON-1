================================================================================
                            KEY DESIGN PRINCIPLES
================================================================================

1. 8PT GRID FOUNDATION
   All spacing must be multiples of 8px for visual consistency

2. WCAG AA COMPLIANCE
   4.5:1 contrast ratio minimum for accessibility

3. SEMANTIC HTML
   Proper structure for accessibility, SEO, and maintainability

4. CSS VARIABLES
   Single source of truth for colors and easy theme switching

5. MOBILE-FIRST
   Design for mobile first, then enhance for larger screens

6. COMPONENT STATES
   Every component has clearly defined states (hover, focus, active, disabled)

7. PROFESSIONAL STANDARDS
   Aligned with industry best practices (Google Material Design, Apple HIG)

8. ACCESSIBILITY FIRST
   Build accessibility in from the start, not as an afterthought
